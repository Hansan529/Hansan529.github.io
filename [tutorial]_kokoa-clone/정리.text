## 11.17 시작
브라우저는 모든 것을 지정해줘야한다.
상단 고정메뉴, 타이틀, 링크, 텍스트그룹, 텍스트, 사이드바 모든 것을 지정해줘야함.
그것을 지정하고 class나 id 메서드로 저장하고, 저장한 값을 불러와서 해당 그룹에 대한
스타일을 적용하고 위치를 이동하고 모든 것을 정해줘야한다. viewport의 크기에 따라서(반응형) 
내부 content들이 스타일이 변화되거나, 접속 기기에 따라서 나타나는 웹페이지를 다르게(적응형) 하던지를 하나하나 정해야함.
HTML : 브라우저에게 웹사이트의 content가 무엇인지 알려줌
CSS : content를 어떻게 보여줘야 할지 정해줌
HTML & CSS = Markup Language, JS Programing Language

폴더,스크립트들은 소문자로만 작성하자. index.html / style.css / app.js
